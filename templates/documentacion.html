Documentación de la API de Animales

Esta API permite obtener información sobre diferentes tipos de animales y sus sonidos.

Endpoints disponibles:

1. Obtener la lista de animales

GET /animales

Devuelve un JSON con los animales y sus sonidos.

    {
        "boa_constrictor": "Tsss!",
        "gato": "Miau Miau!",
        "huron": "Eek Eek!",
        "perro": "Guau Guau!"
    }

2. Obtener información de un animal

GET /animales/<nombre>

Devuelve el nombre y el sonido del animal especificado.

Ejemplo de consulta:

    GET /animales/gato

Ejemplo de respuesta:

    {
        "nombre": "Whiskers",
        "sonido": "Miau Miau!"
    }

Códigos de respuesta:

  • 200 OK - La solicitud se realizó correctamente.
  • 404 Not Found - El animal solicitado no existe en la base de datos.

Para más información, contacta al desarrollador.
